# Reference Files

This directory contains archived reference files that are no longer actively used in the codebase but are kept for reference purposes.

## Contents

### `v0-ui/`
Original V0 UI design system. This was the source of truth for the IKIDO design system before migration to V2. Components from this directory have been migrated to `components/ikido/`.

**Status**: READ-ONLY - Do not modify. Used only as reference for design patterns.

### `iKidO-Dev-Tasks-Reset.postman_collection.json`
Postman collection for development/testing purposes. Contains endpoints for resetting the tasks module in development mode.

**Status**: Reference only - Not used in production code.
